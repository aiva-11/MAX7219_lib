# MAX7219_lib
This project implements custom horizontal scrolling of vertically oriented characters on a 32x8 LED matrix using 4 MAX7219 drivers and bare-metal C programming on an AVR microcontroller.

Connections:
Vcc-5V 

Gnd-Gnd

Clk->A0

DIN->A8

CS->22
